Turing Test

Test that assess a machine's capacity to display human-like intelligence and behavior.

This test was founded by Alan turing in 1950, whose name is based on this test.

Single and Multi Agent

A single-agent system operates independently to accomplish specific tasks. A multi-agent AI system consists of multiple AI entities that work collaboratively to solve complex problems. A multi-agent can be cooperative like a contra game, or competitive like a street fighter game.

Fully- Observable and Partially Observable

In fully-observable environments, an agent can observe every aspects of the environment compared to partially- observable environments where an agent can only observe some aspects of the environment.

Deterministic and Stochastic

In a deterministic environment, an agent can predict the possible outcome of it's actions. In a stochastic environment, unexpected event can happen at any time.

Static and Dynamic

An agent is said to be static when the environment remains unchanged but actions are still being carried out by the agent. Otherwise, the agent is dynamic.

Constraint Satisfaction Problem

Mathematical problem where values must be assigned to variables and all constraints are to be satisfied.

Unary Constraints

Only applies to a single variable eg X must be 3, Y can't be 18, etc.

Binary Constraints

Involves two variables eg these two cells must be divisible by 3.

Higher-order Constraints

Involves three or more variables eg each row in Sudoku must have all numbers from 1 to 9 without repetition.

Large Language Model (LLM)

A machine learning model designed for natural language processing tasks such as language generation.
